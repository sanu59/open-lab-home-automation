\# Open Lab – Voice-Controlled Home Automation



This project is a smart home automation system that lets you control devices (lights, fans, etc.) using voice commands and a web dashboard. It combines:



\- A \*\*React\*\* frontend

\- A \*\*Node.js / Express / Socket.IO\*\* backend

\- \*\*Raspberry Pi\*\* firmware (Python) controlling hardware via GPIO/SPI/LCD
